# 低代码平台
使用Vue3和Sortable.js实现的拖拽组件低代码平台
## 安装
* clone仓库
  ```bash
  git clone https://github.com/gjssss/openai-low-code
  ```
* 安装依赖
  ```bash
  pnpm i
  ```
  > 注意：如果没有安装pnpm可以使用`npm install -g pnpm`安装
* 运行
  ```bash
  pnpm dev
  ```
  运行后，在浏览器中输入`http://localhost:8081`访问
